# Catholic Bible Text Repository

This repository currently contains a single text file:

- `bible.txt` (header indicates "CPDV - Catholic Public Domain Version")

## Current status

A quick structural check of `bible.txt` finds 66 books by unique `Book 1:1` entries.

For a full Catholic canon, you generally expect 73 books, including deuterocanonical books such as:

- Tobit
- Judith
- Wisdom
- Sirach (Ecclesiasticus)
- Baruch
- 1 Maccabees
- 2 Maccabees
